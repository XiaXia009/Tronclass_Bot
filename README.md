# Tronclass_Bot
可以在各Tronclass變體使用 如:虎尾科技大學Ulearn  
該程式掛載在discord機器人上  
目前僅支援數字點名  

## 或許你想看看怎麼樣
### 可以使用的指令
```
/set_channel 設定頻道
/ulearn_login 登入自己的帳號
/test_my_account 在機器人測試帳號登入
/del_my_account 讓機器人解綁帳號
/ulearn_sign_in 簽到
/shut_up 使用人與被使用人各有50%機率被禁言10秒
/課表 顯示課表
/教學問卷 NFU 教學問卷連結
/反饋 
/修復 執行此指令有機會修復登入失敗
/下節甚麼課 
```
